Git Commands!

Git Commando Beskrivning Användning
git rm [file] Tar bort en fil både från din arbetskatalog och Git-indexet Används om du vill ta bort en fil helt och hållet från Git och din lokala dator
git mv [gammalFil] [nyFil] Flytta eller ge nytt namn till en fil Används om du vill byta namn på en fil, tar bort den gamla och lägger till den nya filen i git-indexet
git mergetool Öppnar ett verktyg för att hantera en konflikt visuellt vid en merge Används om det uppstår en konflikt vid en merge. Efter konflikten är löst kan man fortsätta sin merge
git fetch [remote] Hämtar nya ändringar från det angivna fjärrrepositoryt, men ändrar inte i din lokala Används för att kunna granska ändringar och välja vad du vill använda
git show Visar information om den senaste commiten, (kan även lägga till ett commit-ID för att se en specifik commit) Används för att granska förändringar och se historiken i ett Git-repository
