Git Commando Beskrivning Användning
git init Initierar ett nytt Git-repositorium i den aktuella mappen Används för att skapa ett nytt Git-repositorium från grunden. Använd detta kommando när du vill börja spåra ändringar i ett nytt projekt
git clone Klonar ett befintligt Git-repositorium från en angiven URL. Används för att skapa en kopia av ett befintligt Git-repositorium från en fjärrplats. Detta är användbart när du vill arbeta med en existerande kodbas.
